# Лендинг для музея японской анимации `J-Sur`

Проект представляет собой фронтенд часть лендинга для музея японской анимации.

## Стек технологий

* **HTML**
* **CSS**
* **SASS**
* **JS**

## Установка

```
npm ci
npm run build
```
